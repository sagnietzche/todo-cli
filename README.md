# todo-cli

an simple todo cli written in Rust to understand writing CLI applications in Rust.

## sample

```todo list
# TO DO

* Write podts 
* Buy milk
* Have fun

# DONE 

* Feed the cat 

$ todo add "Update CV"
$ todo mark-done "Buy milk"
$ todo list 

# TO DO

* Write post 
* Have fun 
* Update CV 

# DONE 

* Feed the cat 
* Buy milk

```
